Eğer şu alttaki komutu işlersen, 'git branch' komutunu kullanmadan
direkt olarak istediğin yeni dalı oluşturmuş ve o dala
atlamış olursun.

git checkout -b <yeni dalın ismi>

Örneğin:
git checkout -b seyit4
