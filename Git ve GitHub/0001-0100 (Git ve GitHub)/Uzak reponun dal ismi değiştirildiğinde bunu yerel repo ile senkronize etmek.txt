git branch -m master main
git fetch origin
git branch -u origin/main main
git remote set-head origin -a







Yukarıdaki örnekte, birinci satırdaki 'master' ve 'main' kelimeleri değişken.
Uzak repodaki dalın adı 'master' idi ve 'main' yapıldı (GitHub üzerinden yapıldı).
Şimdi yerel reponun da dal ismini 'master' iken 'main' haline getireceğiz.
